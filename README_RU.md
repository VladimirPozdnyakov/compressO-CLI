# CompressO CLI

Быстрый, безопасный и эффективный инструмент сжатия видео на основе FFmpeg с поддержкой нескольких языков.

> Форк [CompressO](https://github.com/codeforreal1/compressO) - переделан из GUI в CLI с расширенными улучшениями безопасности и производительности.

## 🌍 Языки

CompressO CLI поддерживает несколько языков:

- 🇬🇧 **English** (по умолчанию)
- 🇷🇺 **Русский** (Russian)

Приложение автоматически определяет язык на основе аргументов командной строки или предлагает пользователю выбрать язык при первом запуске.

## ✨ Особенности

### Основной функционал
- 🎯 **Интерактивный режим** - Перетаскивание видео или запуск без аргументов для пошаговой работы
- 🌐 **Многоязычная поддержка** - Выбор между английским и русским языками
- ⚡ **Пакетная обработка** - Сжатие нескольких видео или целых каталогов одновременно
- 🎚️ **Умные пресеты** - Выбор между Thunderbolt (5x скорость) или Ironclad (качество)
- 📊 **Реальное время прогресса** - Скорость кодирования в реальном времени (fps), ETA и индикатор прогресса
- 🔍 **Информация о видео** - Подробное отображение метаданных с оценкой размера файла
- 🎬 **Конвертация формата** - Поддержка MP4, MOV, WebM, AVI, MKV, M4V, WMV, FLV
- 🔄 **Преобразования видео** - Поворот, отражение, обрезка, изменение размера, изменение FPS
- 📦 **Вывод в формате JSON** - Машиночитаемый вывод для автоматизации
- 🎨 **Красивый терминальный интерфейс** - Цветной вывод с символами Unicode

### Функции безопасности
- 🔒 **Защита от перехода по путям** - Проверяет и очищает все пути к файлам
- 🔗 **Предотвращение атак через символические ссылки** - Преобразует символические ссылки в реальные пути
- 🛡️ **Защита системных каталогов** - Предотвращает запись в чувствительные места
- 🔐 **Безопасное разрешение FFmpeg** - Настраиваемый доверенный путь к FFmpeg
- ⚛️ **Защита TOCTOU** - Атомарные операции с файлами устраняют гонки условий
- 🧹 **Гарантированная очистка** - RAII гарантирует, что временные файлы всегда удаляются
- 🔍 **Очистка путей** - Удаляет опасные символы и последовательности
- 📝 **Журнал безопасности** - Журнал аудита операций с путями и предупреждений

### Оптимизации производительности
- 🚀 **5x Более быстрое кодирование** - Пресет Thunderbolt использует FFmpeg ultrafast
- ⏱️ **Оптимизированные регулярные выражения** - Кэшированные шаблоны экономят 150 мкс на видео
- 💤 **Эффективное ожидание** - Блокировка ввода-вывода вместо активного опроса
- 📉 **Одиночный запуск FFmpeg** - Исключает избыточные чтения метаданных
- 📦 **Минимальный бинарный файл** - 2.3 МБ упакованный релизный билд

## 🚀 Установка

### Предварительные требования

**FFmpeg** требуется и должен быть установлен:

| Платформа | Команда установки |
|----------|---------------------|
| Windows  | `winget install FFmpeg` |
| macOS    | `brew install ffmpeg` |
| Ubuntu/Debian | `sudo apt install ffmpeg` |
| Fedora/RHEL | `sudo dnf install ffmpeg` |
| Arch Linux | `sudo pacman -S ffmpeg` |

Проверьте, что FFmpeg установлен:
```bash
ffmpeg -version
```

## ⚡ Быстрый старт

### Самое простое использование

```bash
# Интерактивный режим - просто запустите его (запрашивает выбор языка)
compresso

# Или перетащите файл видео на compresso.exe
```

### Командная строка

```bash
# Сжатие видео с настройками по умолчанию
compresso video.mp4

# Быстрое сжатие (пресет Thunderbolt)
compresso video.mp4 -p thunderbolt -q 70

# Лучшее качество (пресет Ironclad)
compresso video.mp4 -p ironclad -q 85

# Конвертация формата при сжатии
compresso video.mp4 output.webm -f webm
```

```bash

# Несколько файлов
compresso video1.mp4 video2.mp4 video3.mp4 -q 70

# Весь каталог

# Вывод JSON для скриптов
```

## 🎚️ Пресеты

CompressO предлагает два тщательно настроенных пресета, оптимизированных для разных случаев использования:

### ⚡ Thunderbolt (Приоритет скорости)

**Лучше для:** Быстрого сжатия, черновиков, социальных сетей, записей экрана

**Производительность:**
- Кодирование в 5 раз быстрее, чем средний пресет
- 1-минутное видео: ~6 секунд времени кодирования
- Пакет из 10 видео: ~1 минута

**Технические детали:**
- Пресет FFmpeg: `ultrafast`
- Настройка: `fastdecode` (оптимизированное воспроизведение)
- Кодек: H.264 (libx264)
- Размер файла: на 10-20% больше, чем Ironclad при том же качестве

**Когда использовать:**
- ✅ Рабочие процессы с ограничением по времени
- ✅ Обработка черновиков
- ✅ Быстрый обмен
- ✅ Записи экрана
- ✅ Социальные сети (платформа всё равно перекодирует)
- ❌ Не для: Архивации, финальной доставки, оптимизации хранения

### 🛡️ Ironclad (Приоритет качества)

**Лучше для:** Архивации, профессиональной доставки, оптимизации хранения

**Производительность:**
- Лучшее соотношение качества к размеру
- 1-минутное видео: ~30 секунд времени кодирования
- Максимальная эффективность сжатия

**Технические детали:**
- Пресет FFmpeg: `slow`
- Кодек: H.264 (libx264)
- Дополнительные флаги: `-pix_fmt yuv420p`, `-movflags +faststart`
- Размер файла: Базовый уровень (наименьший при том же качестве)

**Когда использовать:**
- ✅ Архивное сжатие
- ✅ Профессиональная доставка
- ✅ Финальные экспортные работы
- ✅ Оптимизация хранения
- ✅ Максимальное сохранение качества
- ❌ Не для: Быстрых превью, рабочих процессов с ограничением по времени

### Сравнение пресетов

| Аспект | Thunderbolt | Ironclad |
|--------|-------------|----------|
| Скорость кодирования | **5x быстрее** ⚡ | 1x базовый уровень |
| Размер файла | +10-20% больше | Базовый уровень (наименьший) |
| Визуальное качество | Идентично (тот же CRF) | Идентично (тот же CRF) |
| Воспроизведение | Оптимизировано 🚀 | Стандартное |
| Сценарий использования | Приоритет скорости | Приоритет качества/размера |

## 📚 Примеры

### Базовое сжатие

```bash
# Настройки по умолчанию (пресет Ironclad, качество 70)
compresso video.mp4

# Указать качество (0-100, выше = лучше)
compresso video.mp4 -q 85

# Быстрое сжатие с Thunderbolt
compresso video.mp4 -p thunderbolt -q 70

# Вывод в конкретный файл
compresso video.mp4 output.mp4

# Использовать русский язык
compresso video.mp4 --language russian

# Использовать английский язык
compresso video.mp4 --language english
```

### Конвертация формата

```bash
# Конвертировать в WebM
compresso video.mp4 output.webm -f webm

# Конвертировать в MOV
compresso video.avi video.mov -f mov

# Автоопределение формата по расширению
compresso video.mp4 output.webm
```

### Преобразования видео

```bash
# Изменить размер до 720p
compresso video.mp4 --width 1280 --height 720

# Изменить частоту кадров
compresso video.mp4 --fps 30

# Повернуть на 90 градусов по часовой стрелке
compresso video.mp4 --rotate 90

# Отразить по горизонтали
compresso video.mp4 --flip-h

# Отразить по вертикали
compresso video.mp4 --flip-v

# Обрезка (ширина:высота:x:y)
compresso video.mp4 --crop 1920:1080:0:0
```

### Параметры аудио

```bash
# Удалить аудиодорожку
compresso video.mp4 --mute

# Сохранить аудио (по умолчанию)
compresso video.mp4
```

### Расширенные примеры

```bash
# Комбинирование нескольких опций
compresso video.mp4 \
  -q 80 \
  -p ironclad \
  --width 1920 \
  --height 1080 \
  --fps 30 \
  --rotate 90 \
  -y

# Быстрое превью 720p
compresso large-4k-video.mp4 preview.mp4 \
  -p thunderbolt \
  -q 60 \
  --width 1280 \
  --height 720 \
  --fps 30

# Архивное сжатие (максимальное сохранение качества)
compresso raw-footage.mov archive.mp4 \
  -p ironclad \
  -q 90 \
  --width 3840 \
  --height 2160 \
  --fps 60

# Оптимизация для социальных сетей
compresso video.mp4 instagram.mp4 \
  -p thunderbolt \
  -q 70 \
  --width 1080 \
  --height 1920 \
  --fps 30
```

### Информация о видео

```bash
# Показать подробную информацию о видео
compresso video.mp4 --info

# Вывод JSON для скриптов
compresso video.mp4 --info --json
```

### Пакетная обработка

```bash
# Обработать все MP4 файлы в текущем каталоге
compresso *.mp4 -q 70 -p thunderbolt

# Обработать весь каталог
compresso --dir ./raw-videos -q 80

# Обработать с подробным выводом
compresso *.mp4 -q 70 -v
```

## 🎛️ Справочник опций

### Ввод/вывод

| Опция | Описание | Пример |
|--------|-------------|---------|
| `<INPUT>` | Входной видеофайл(ы) | `video.mp4` или `*.mp4` |
| `<OUTPUT>` | Путь к выходному файлу (опционально) | `output.mp4` |
| `--dir <DIR>` | Обработать все видео в каталоге | `--dir ./videos` |

### Настройки сжатия

| Опция | Короткая | Описание | По умолчанию |
|--------|-------|-------------|---------|
| `--quality <Q>` | `-q` | Качество (0-100, выше = лучше) | `70` |
| `--preset <P>` | `-p` | Пресет: `thunderbolt` или `ironclad` | `ironclad` |
| `--format <F>` | `-f` | Выходной формат: mp4, mov, webm, avi, mkv, m4v, wmv, flv | (авто) |

### Обработка видео

| Опция | Описание | Пример |
|--------|-------------|---------|
| `--width <W>` | Выходная ширина видео в пикселях | `--width 1920` |
| `--height <H>` | Выходная высота видео в пикселях | `--height 1080` |
| `--fps <FPS>` | Выходная частота кадров | `--fps 30` |
| `--mute` | Удалить аудиодорожку | `--mute` |

### Преобразования

| Опция | Описание | Значения |
|--------|-------------|--------|
| `--rotate <DEG>` | Повернуть видео | `90`, `180`, `270`, `-90`, `-180`, `-270` |
| `--flip-h` | Отразить по горизонтали | (флаг) |
| `--flip-v` | Отразить по вертикали | (флаг) |
| `--crop <CROP>` | Обрезать видео | `WxH:X:Y` или `W:H:X:Y` |

### Поведение

| Опция | Короткая | Описание |
|--------|-------|-------------|
| `--language <LANG>` | | Язык интерфейса: `english` или `russian` |
| `--overwrite` | `-y` | Перезаписать выходной файл без подтверждения |
| `--verbose` | `-v` | Показать подробный вывод FFmpeg |
| `--json` | | Вывести результаты в формате JSON |
| `--info` | | Показать только информацию о видео (без сжатия) |

### Справка

| Опция | Короткая | Описание |
|--------|-------|-------------|
| `--help` | `-h` | Показать справочное сообщение |
| `--version` | `-V` | Показать версию |

## 🔒 Конфигурация безопасности

CompressO реализует несколько уровней безопасности для защиты от атак.

### Переменные окружения

#### `COMPRESSO_FFMPEG_PATH`

Укажите доверенный путь к бинарному файлу FFmpeg (рекомендуется для продакшена):

```bash
# Linux/macOS
export COMPRESSO_FFMPEG_PATH=/usr/bin/ffmpeg

# Windows (PowerShell)
$env:COMPRESSO_FFMPEG_PATH = "C:\Program Files\ffmpeg\bin\ffmpeg.exe"

# Windows (CMD)
set COMPRESSO_FFMPEG_PATH=C:\Program Files\ffmpeg\bin\ffmpeg.exe
```

**Когда использовать:**
- ✅ Продакшн-среды
- ✅ CI/CD пайплайны
- ✅ Общие серверы
- ✅ Рабочие процессы с повышенной безопасностью

#### `COMPRESSO_FFMPEG_VERIFY`

Включить строгую проверку встроенного FFmpeg:

```bash
export COMPRESSO_FFMPEG_VERIFY=1
```

**Выполняемые проверки:**
- Файл исполняемый (Unix)
- Бинарный файл отвечает на `--version`
- Вывод содержит "ffmpeg version"

### Приоритет разрешения FFmpeg

1. **`COMPRESSO_FFMPEG_PATH`** (явный, наиболее безопасный)
2. **Встроенный FFmpeg** (каталог приложения, при необходимости проверяется)
3. **Системный PATH** (наименее безопасный, регистрирует предупреждение)

### Предупреждения безопасности

CompressO регистрирует события, связанные с безопасностью:

```
⚠ Предупреждение: входной путь содержит '..', что может указывать на переход по путям
ℹ входной путь разрешён через символическую ссылку:
  Предоставлено: link.mp4
  Разрешено: /real/path/video.mp4
Ошибка: Отказ от записи в системный каталог: /etc/output.mp4
```

**Всегда просматривайте предупреждения безопасности** в stderr выводе.

### Защищенные системные каталоги

CompressO предотвращает запись в:

**Unix/Linux:**
- `/etc/` - Системная конфигурация
- `/sys/` - Интерфейсы ядра
- `/proc/` - Информация о процессах
- `/dev/` - Файлы устройств
- `/boot/` - Загрузочные файлы
- `/root/` - Домашняя папка root
- `/` - Корневая файловая система

**Windows:**
- `C:\Windows\` - Системные файлы
- `C:\Program Files\` - Приложения
- Корни дисков (`C:\`, `D:\`, и т.д.)

## 📊 Руководство по качеству

### Значения качества и сопоставление CRF

| Качество | CRF | Размер файла | Сценарий использования |
|---------|-----|-----------|----------|
| 100 | 24 | Наибольший | Почти без потерь, архивирование |
| 90 | 25 | Очень большой | Профессиональная доставка |
| 80 | 26 | Большой | Высокое качество обмена |
| **70** | **28** | **Средний** | **Сбалансированный (по умолчанию)** |
| 60 | 30 | Маленький | Хорошее сжатие |
| 50 | 31 | Меньше | Заметное сжатие |
| 40 | 33 | Очень маленький | Значительная потеря качества |
| 30 | 34 | Крошечный | Тяжелое сжатие |

**Меньше CRF = Выше качество = Больше размер файла**

### Выбор качества

**Общие рекомендации:**

- **90-100:** Архивация, профессиональное мастерингование
  - Минимальная потеря качества
  - Большой размер файла
  - Используйте пресет Ironclad

- **75-85:** Профессиональная доставка, YouTube
  - Отличное качество
  - Разумный размер файла
  - Используйте пресет Ironclad

- **60-75:** Общий обмен, загрузка в интернет
  - Хорошее качество
  - Хорошее сжатие
  - Подходит любой пресет

- **40-60:** Быстрый обмен, устройства с низкой производительностью
  - Заметное сжатие
  - Маленький размер файла
  - Используйте пресет Thunderbolt

- **Ниже 40:** Только экстренное сжатие
  - Видимые артефакты
  - Очень маленькие файлы
  - Использовать с осторожностью

### Влияние пресета на размер файла

При том же значении качества:

| Качество | Ironclad | Thunderbolt | Разница |
|---------|----------|-------------|------------|
| 70 | 10.0 MB | 11.5 MB | +15% |
| 80 | 15.0 MB | 17.3 MB | +15% |
| 90 | 25.0 MB | 28.8 MB | +15% |

**Примечание:** Thunderbolt создаёт немного большие файлы, но кодирует в 5 раз быстрее.

## 🐛 Устранение неполадок

### Распространенные проблемы

**1. "FFmpeg не найден"**
```
Решение: Установите FFmpeg и убедитесь, что он находится в PATH
- Windows: winget install FFmpeg
- macOS: brew install ffmpeg
- Linux: sudo apt install ffmpeg

Или установите явный путь:
export COMPRESSO_FFMPEG_PATH=/usr/bin/ffmpeg
```

**2. "Файл не найден" или "Неверный ввод"**
```
Решение: Проверьте путь к файлу и права доступа
- Используйте абсолютные пути или заключайте пути с пробелами в кавычки
- Проверьте существование файла: ls -la video.mp4
- Проверьте, что файл является действительным видео: file video.mp4
```

**3. "Отказано в доступе" на вывод**
```
Решение: Проверьте права доступа к каталогу вывода
chmod +w output-directory
```

**4. "Отказ от записи в системный каталог"**
```
Решение: Это функция безопасности
- Не указывайте вывод в системные каталоги
- Используйте текущий каталог или домашний каталог
```

**5. Слишком медленное сжатие**
```
Решение: Используйте пресет Thunderbolt
compresso video.mp4 -p thunderbolt -q 70
```

**6. Выходной файл слишком большой**
```
Решение: Понизьте качество или используйте Ironclad
compresso video.mp4 -p ironclad -q 60
```

**7. Предупреждение о том, что путь содержит '..'**
```
Решение: Используйте абсолютные пути
compresso /full/path/to/video.mp4
```

Для отладки используйте подробный режим:
```bash
compresso video.mp4 -q 70 -v
```
